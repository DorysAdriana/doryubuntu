Selectores en CSS3

El objetivo de esta pagina es mostrar el uso de los selectores en Css3, para que sirven, y cuales son sus tipos asi que no te despeques de aqui!!

que es un selector?

Un selector permite seleccionar un elemento web para de esta forma poder darle estilo, es decir mejorar su estructura a nuestra conveniencia.

Tipos de Selectores

los selectores se clasifican de la siguente manera:

* Selectores Universales
* Selectores de hijos
* Selector Adyacente
* Selector de Atributos

Selector Universal

Este selector se encarga de seleccionar todos los elementos de una pagina
normalmente hay que personalizar los margenes y tipo de letra entre otras cosas.
en este caso en el selector universal utilizamos la alineacion centrada del texto el tipo de letra fantasy y el estilo de letra italic
* { text-align: center; font-style: italic; }

Selector de hijos

Referencia el elemento de la derecha cuando tiene el elemento de la izquierda como su padre
En este caso usamos el div que lo precede como padre del elemento font con el cual puedo mostrar parte del codigo usado en css.
lo que hace es cambiarle el color al teexto y cambiarle el estilo para que resalte un poco. div font { color: crimson; font-style: normal; }

Selector Adyacente

Este selector se encarga de referenciar los elementos que son hermanos la
referencia apunta al elemento de la derecha cuando
este inmediatamente precedido por el de la izquierda
En este caso usamos
h2+p { font-size: 0.5cm; }

Selector de atributos

Hay varios tipos de selectores de atributos por ahora solo mostrare los 4 primeros tipos de selectores de atributos:

.[nombre_elemento]
.[nombre_atributo = valor]
.[nombre_atributo~=valor]
.[nombre_atributo|=valor]

.[nombre_elemento]


Selecciona los elementos que tienen establecido el atributo llamado "nombre elemento"

.[nombre_atributo = valor]


Selecciona los elementos que tienen establecido un atributo llamado atributo con un valor igual a valor

.[nombre_atributo~=valor]


Selecciona los elementos con un atributo llamado "nombre_atributo" y su valor es una lista de palabras que es exactamente igual al valor

.[nombre_atributo|=valor]


Selecciona los elementos que tienen un atributo llamado "nombre_atributo" y su valor es una serie de palabras separadas por guiones comenzando con un valor y solo son validas para tipo lang
los selectores de atributos usados se pueden ver reflejados en las tablas en los cuales se usa para darle color y tipo de texto,fondo y bordes distintos. .table1 { font-style: normal; background-color: lightcoral; width: 300px; height: 30px; border: 2px solid black; }
.table2 { width: 300px; height: 30px; background-color: orange; border: 3px solid orangered; }
.table3 { font-style: normal; width: 300px; height: 30px; background-color: crimson; border: 1px solid darkviolet; }
